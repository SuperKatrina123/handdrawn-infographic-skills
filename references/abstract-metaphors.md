# Abstract Concepts → Visual Metaphors

When the source text uses an **abstract state verb / noun** (迷路, 稀释, 碰撞, 挤压, 失效, 漂移, 污染, 坍塌, 内耗…), do NOT write the word onto the card. Draw the metaphor instead. The drawing carries the meaning; at most ONE italic handwritten caption under the drawing is allowed as a gloss.

## Trigger Vocabulary (scan input for these)

Any of these words in the source text should prompt you to open this table and find the canonical visual:

迷路 / 找不到 / 决策困难 / 选不出 · 稀释 / 分散 / 摊薄 · 碰撞 / 混淆 / 撞车 / 分不清 · 挤压 / 爆仓 / 装不下 / 溢出 · 失效 / 过期 / 陈旧 / 击穿 · 漂移 / 偏离 / 跑偏 · 污染 / 串味 / 掺杂 · 坍塌 / 崩塌 / 级联 · 退化 / 变钝 / 生锈 · 内耗 / 拉扯 · 黑箱 / 不透明 · 过载 / 超载

## Rules for Using This Table

1. **Scan first.** Before drafting the ASCII layout, grep the input text for the trigger vocabulary above. For each match, decide whether it deserves its own card or is background context.

2. **One metaphor per card.** Don't stack "迷路 + 稀释 + 碰撞" into one drawing. Each abstract concept gets its own cell so the visual reads cleanly.

3. **The only text allowed is ONE italic caption** underneath the drawing, describing the *consequence* (not naming the concept). Good caption = *"光被分给太多人，每个人都没被照亮。"* Bad caption = *"注意力稀释"*.

4. **If the concept is not in the table,** invent a metaphor using the same rules: concrete object, everyday scene, line-art only. Then add the new row to this file so future runs reuse it.

5. **Never write the abstract word as a title.** Titles like "注意力稀释" / "预算挤压" / "缓存失效" on cards are banned. The drawing is the title.

6. **Combine with character cameos.** A stick-figure reaction (sweat drop, shrug, `?` above head, slumped posture) often sells the metaphor better than any label. Pair the object-metaphor with a tiny human reaction whenever it fits.
